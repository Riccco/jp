# OS
> オペレーティングシステム

ハードウェアやアプリケーションソフトウェアを管理、制御するソフトです。

# 制御プログラム

ハードウェア資源の状態を常に監視し、コンピュータの効率的な利用を実現するソフトウェア群です。

* ジョブ管理
* タスク管理
* 記憶管理
* ファイル管理
* その他

# OSS
> オープンソースソフトウェア(oss Open Source Software)
