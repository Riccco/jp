# CPU

## クロック

クロック信号「クロックパルス」は：各装置の動作のタイミングを合わせるためにCPUが発する周期的な信号です。

クッロク信号の高低のサイクルが一秒間に繰り返される回数をクッロク周波数といい

## バス

コンピュータ内部でデータをやりとりするための伝送路です。
